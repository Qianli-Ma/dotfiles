# dotfiles
[autoupdate.sh](https://github.com/Qianli-Ma/dotfiles/blob/master/setup.sh) auto updates all Brew formulas and all github repos on macOS for now

Linux setup is split into focused scripts:
- `linux/etc.sh` for distro mirrors and base packages
- `linux/homebrew.sh` for Linuxbrew, Homebrew mirrors, `fzf`, and Brewfile installs
- `linux/oh-my-zsh.sh` for Oh My Zsh, plugins, and theme
- `linux/dotfiles/` for tracked Linux shell config files such as `.Brewfile`, `.zshrc`, and `.p10k.zsh`
